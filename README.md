Spring-Project
===============
### 프로젝트 소개
> 일반 네이버 블로그나 티스토리와 같이 블로그를 생성하고 관리하는 웹

### 프로젝트 목적
> 스프링 사용과 기본 원리를 익히기 위해 프로젝트 시작

### 사용된 언어 및 툴
- HTML / CSS / JavaScript
- JSTL / JSON / AJAX 
- 스프링(4.3.3) / Java
- 오라클 / Mybatis

### 기능
- Register / Login
- Blog
- Blog Admin (User) / Blog Admin (Admin)
- Blog Search
